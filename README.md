# Code Alpha Internship Projects

Welcome to my Code Alpha internship repository! 🚀 This repo contains the projects I’ve worked on during my internship, including web applications built with HTML, CSS, and JavaScript.

## 📂 Project 1: Age Calculator

This is a simple, web-based age calculator where users can enter their date of birth and find out their exact age in years, months, and days.

### 🛠️ Tech Stack
- **HTML**: Structure of the web page  
- **CSS**: Styling and layout  
- **JavaScript**: Age calculation logic  

### ✨ Features
- Calculates precise age from date of birth  
- Simple and user-friendly interface  
- Real-time results on button click

## 📂 Project 2: To-Do List Web App

A sleek and interactive to-do list app to help users manage and prioritize their daily tasks.

### 🛠️ Tech Stack
- **HTML**: Page layout and input elements  
- **CSS**: Modern design with glassmorphism effect  
- **JavaScript**: Task creation, deletion, completion, and local storage  

### ✨ Features
- Add, edit, and delete tasks  
- Mark tasks as complete/incomplete  
- Persist tasks using local storage  
- Beautiful background and glass-style container
